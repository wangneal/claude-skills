# 读取数据服务

> 来源: https://dev.kingdee.com/open/detail/sdk/2377816509545198592

1. 读取数据服务介绍

读取数据服务类（kd.bos.servicehelper.BusinessDataServiceHelper），提供批量、单条读取数据方法。

BusinessDataServiceHelper服务取出的数据是有结构的，分为单据头、单据体、子单据体，逐层往下。 QueryServiceHelper 服务取出的数据是拉平的，会关联单据头、单据体、子单据体，输出拉平的数据。

1.1 方法介绍

数据操作服务类提供多种不同场景的数据操作接口，下面列举一些常用的方法。

序号	方法名	方法说明	参数说明
1	newDynamicObject(String entityName)	通过实体标识创建一个动态实体数据包	entityName：实体标识
2	loadSingle(Object pk, String entityName)	读取单张实体数据	pk：主键
entityName：实体标识
3	loadSingleFromCache(Object pk, String entityName)	读取单张实体数据，使用缓存	pk：主键
entityName：实体标识
4	load(Object[] pkArray, IDataEntityType type)	批量读取实体数据	pkArray：主键数组
type：实体类型
5	loadFromCache(Object[] pkArray, DynamicObjectType type)	批量读取实体数据，使用缓存	pkArray：主键数组
type：实体类型
6	load(String entityName, String selectProperties, QFilter[] filters)	按条件读取实体数据	entityName：实体标识
selectProperties：读取字段：使用逗号分隔字段标识
filters：过滤条件
...	...	...	...
1.2 方法案例

列举一些常用方法的简单用例介绍。

1.2.1 1）、newDynamicObject(String entityName)

通过实体标识创建动态数据包，可以对数据包进行赋值等操作。

// 通过实体标识（test_bill）创建对应实体数据包
DynamicObject dObject = BusinessDataServiceHelper.newDynamicObject("test_key");
dObject。set("propertyName","value");
1.2.2 2）、loadSingle(Object pk, String entityName)

通过主键与实体标识获取某一张单的实体数据，对其进行操作。

DynamicObject formInfo = BusinessDataServiceHelper.loadSingle("pk", "test_key");
String billno = formInfo.getString("billno");
1.2.3 3）、load(String entityName, String selectProperties, QFilter[] filters)

根据对应参数获取符合条件的相应实体数据，对其进行批量处理等操作。

QFilter filter =  new QFilter("id", QCP.equals, id);
QFilter[] filters1 = new QFilter[]{filter};
DynamicObject[] forms = BusinessDataServiceHelper.load("test_key", "id,name,number,entity.id", filters);
