1. 数据删除服务介绍

数据删除服务类（kd.bos.servicehelper.operation.DeleteServiceHelper），主要提供单据删除服务相关接口。
SDK：https://dev.kingdee.com/sdk/Cosmic%20V5.0.002/index.html?nav=class

1.1 方法介绍
序号	方法名	方法说明	参数说明
1	delete(IDataEntityType type,Object[] pks)	根据实体类型和实体ID删除数据：直接删库，不走删除校验	type 实体类型
pks 实体ID
2	deleteOperate(String operationKey, String entityName, Object[] pks, OperateOption option)	调用单据删除操作服务：执行删除校验及插件，校验通过后才删库	operationKey 操作代码
entityName 实体名
pks 实体ID
option 自定义参数
3	delete(String entityName, QFilter[] qFilters)	根据实体名称和过滤条件删除数据：直接删库，不走删除校验	entityName 实体名
qFilters 过滤条件
...	...	...	...
1.2 方法用例

简单介绍相关方法调用。

1.2.1 1）、delete(IDataEntityType type,Object[] pks)

直接删库，不走删除校验，不删除关联数据。

Object[] ids = new Object[]{formId};
IDataEntityType type = EntityMetadataCache.getDataEntityType("test_entity");
DeleteServiceHelper.delete(type, ids);
1.2.2 2）、delete(String entityName, QFilter[] qFilters)

根据实体名称和过滤条件删除数据：直接删库，不走删除校验

QFilter q1 = new QFilter("user", QCP.equals, Long.valueOf(RequestContext.get().getUserId()));
QFilter q2 = new QFilter("formId", QCP.equals, formId);
QFilter[] filter = new QFilter[]{q1, q2};
DeleteServiceHelper.delete("test_entity", filter);
1.2.3 3). deleteOperate (String operationKey, String entityName, Object[] pks, OperateOption option)

调用单据删除操作服务：执行删除校验及插件，校验通过后才删库。（推荐）

DeleteServiceHelper("delete", "test_entity", new Object[] { billId }, OperateOption.create());
