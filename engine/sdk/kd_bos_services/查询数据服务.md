# 查询数据服务

> 来源: https://dev.kingdee.com/open/detail/sdk/2377816508932831232

1. 数据查询服务介绍

数据查询服务类（kd.bos.servicehelper.QueryServiceHelper），提供常用的查询的方法。
SDK：https://dev.kingdee.com/sdk/Cosmic%20V5.0.002/index.html?nav=class

1.1 方法介绍

数据查询服务常用的方法：

序号	方法名	方法说明	参数说明
1	query(String entityName, String selectFields,QFilter[] filters, String orderBys)	查询单据（排序），返回单据头、单据体拉平的数据包	entityName：单据实体标识
selectFields：查询字段，多个字段用逗号隔开
filters：过滤条件
orderBys：排序条件
2	queryDataSet(String algoKey, String entityName, String selectFields, QFilter[] filters,String orderBys, int top)	查询单据（排序，取top条），返回拉平的数据流	algoKey：查询标识，调用者通过指定查询标识来互相区分，后续可用此对本本服务进行归类，分析各调用者的性能表现
entityName：单据实体标识
selectFields：查询字段，多个字段用逗号隔开
filters：过滤条件
orderBys：排序条件
top：查询前top条
3	queryOne(String entityName, String selectFields, QFilter[] filters)	查询一条数据（第一条）	entityName：单据实体标识
selectFields：查询字段，多个字段用逗号隔开
filters：过滤条件
4	queryPrimaryKeys(String entityName, QFilter[] filters, String orderBys, int top)	查询主键（排序，取top条）	entityName：单据实体标识
filters：过滤条件
orderBys：排序条件
top：查询前top条
5	exists(String entityName, Object pkId)	判断单据主键是否存在	entityName：单据实体标识
pkId：主键
6	exists(String entityName, QFilter[] qFilters)	判断符合条件的单据是否存在	entityName：单据实体标识
qFilters：查询条件
