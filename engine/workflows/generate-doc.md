<purpose>
为金蝶开发项目生成中文技术文档。
</purpose>

<available_agent_types>
- kd-doc-writer — 文档生成专家
</available_agent_types>

<process>

## 1. 解析参数

```bash
DOC_TYPE=$1
OUTPUT_FILE=${2:-""}
PROJECT_DIR="."
```

**支持的文档类型**:
- readme - README.md
- api - API 文档
- guide - 开发指南
- changelog - 更新日志

## 2. 扫描项目文件

```bash
# 扫描 Java 文件
JAVA_FILES=$(find "$PROJECT_DIR/src" -name "*.java" 2>/dev/null)

# 扫描配置文件
CONFIG_FILES=$(find "$PROJECT_DIR/config" -type f 2>/dev/null)

# 统计信息
JAVA_COUNT=$(echo "$JAVA_FILES" | wc -l)
```

## 3. 调用文档生成代理

```javascript
Task({
  prompt: `
<doc_request>
**文档类型**: ${DOC_TYPE}
**项目信息**: {
  名称: $(cat .kingdee/config.json | jq -r '.name'),
  描述: $(cat .kingdee/config.json | jq -r '.description'),
  Java文件数: ${JAVA_COUNT}
}
**源代码路径**: ${PROJECT_DIR}/src
**输出路径**: ${OUTPUT_FILE}
</doc_request>
  `,
  subagent_type: "kd-doc-writer",
  description: "生成项目文档"
})
```

## 4. 根据类型生成文档

## 5. 保存文档

```bash
if [[ -n "$OUTPUT_FILE" ]]; then
  # 保存到指定位置
  echo "$DOC_CONTENT" > "$OUTPUT_FILE"
else
  # 使用默认文件名
  case "$DOC_TYPE" in
    readme) OUTPUT_FILE="README.md" ;;
    api) OUTPUT_FILE="docs/API.md" ;;
    guide) OUTPUT_FILE="docs/GUIDE.md" ;;
    changelog) OUTPUT_FILE="CHANGELOG.md" ;;
  esac

  echo "$DOC_CONTENT" > "$OUTPUT_FILE"
fi
```

