# Phase 14: 列表插件支持

## 目标

实现列表插件代码生成功能，支持分页查询、过滤条件、总记录数统计，与现有代码生成器集成。

## 背景

项目已完成表单插件（Phase 10）、工作流插件（Phase 10）、报表插件（Phase 11）的增强实现，现需扩展支持列表插件。

## 任务列表

### Task 1: 创建增强版列表插件模板

**类型**: auto
**文件**: `engine/templates/plugin/ListPlugin-Enhanced.java`

**要求**:
- 继承 AbstractListPlugin
- 实现 query 方法（核心查询逻辑）
- 支持分页查询（分页参数处理）
- 支持过滤条件（QFilter 构建）
- 支持总记录数统计（getTotalCount）
- 集成常量引用（无魔法值）
- 完整异常处理
- 日志记录
- JavaDoc 注释

**参考**: ReportPlugin-Enhanced.java 模板结构

**验证**:
- 模板文件存在
- 包含 query 方法
- 包含分页逻辑
- 包含过滤条件构建
- 包含总记录数统计
