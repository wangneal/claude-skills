# SDK 使用规范 (SDK Usage Standards)

本文档定义金蝶 BOS SDK 的使用最佳实践。

## 1. DynamicObject 操作

### 读取字段值

```csharp
DynamicObject obj = (DynamicObject)this.Model.DataObject;

// ✅ 正确 - 检查字段存在
if (obj.DynamicObjectType.Properties.ContainsKey("fieldname"))
{
    object value = obj["fieldname"];
}

// ✅ 使用泛型方法
string billNo = obj.GetString("billno");
```

### 设置字段值

```csharp
// ✅ 正确 - 类型匹配
obj["amount"] = 100.00m;
obj["status"] = "Approved";

// ✅ 正确 - 使用 SetDynamicObjectItem
obj.SetDynamicObjectItem("modifieddate", DateTime.Now);
```

// ✅ 正确 - 检查单据状态
if (CanStartWorkflow(billNo))

## 3. 权限检查

### 检查操作权限

```csharp
// ✅ 正确 - 使用权限服务
bool hasPermission = PermissionServiceHelper.CheckPermission(
    currentUserId,

if (!hasPermission)

    throw new BusinessException("您没有审批权限");
}
```

## 4. 数据查询

### 使用 QueryService

```csharp
// ✅ 正确 - 参数化查询
var query = new QueryBuilder()
    .Select("billno", "billstatus")
    .From("tableName")
    .Where("billno = @billno")
    .AddParameter("@billno", billNo);

var result = QueryService.Execute(query);
```

## 5. 事务处理

### 使用事务

```csharp
using (var transaction = DBService.BeginTransaction())
{
    try
    {
        // 操作 1
        SaveBillHeader(bill, transaction);

        // 操作 2
        SaveBillEntries(bill, transaction);

        // 提交事务
        transaction.Commit();
    }
    catch
    {
        // 回滚事务
        transaction.Rollback();
        throw;
    }
}
```
