# 代码格式规范 (Code Format Standards)

本文档定义金蝶苍穹开发的代码格式标准。

## 1. 缩进

- 使用 **4 个空格**缩进（不使用 Tab）
- 保持整个项目缩进一致

### 示例

```csharp
public class WorkflowService
{
    public void ProcessBill()
    {
        if (condition)
        {
            // 4 空格缩进
            DoSomething();
        }
    }
}
```

## 2. 大括号

- 使用 **Allman 风格**（大括号独占一行）
- 即使单行语句也使用大括号

### ✅ 正确示例

```csharp
if (isValid)
{
    ProcessData();
}
else
{
    LogError();
}
```

### ❌ 错误示例

```csharp
if (isValid) {
    ProcessData();
}

if (isValid)
    ProcessData();
```

## 3. 空格

### 运算符两侧

```csharp
int result = a + b;  // ✅
int result=a+b;      // ❌
```

### 逗号后

```csharp
Method(param1, param2, param3);  // ✅
Method(param1,param2,param3);    // ❌
```

### 关键字后

```csharp
if (condition)    // ✅
if(condition)     // ❌
```

## 4. 空行

- 方法之间：1 个空行
- 逻辑块之间：1 个空行
- 类成员分组：使用注释分隔

### 示例

```csharp
public class BillService
{
    // Fields

    private string _billNo;

    // Properties

    public string BillNo { get; set; }

    // Methods

    public void ProcessBill()
    {
        // 逻辑块 1
        ValidateData();

        // 逻辑块 2
        SaveData();
    }

    public void ValidateData()
    {
        // 实现
    }
}
```

## 5. 行长度

- 最大行长度：**120 字符**
- 超长行应换行

### 换行规则

```csharp
// 方法调用换行
var result = SomeVeryLongMethodName(
    parameter1,
    parameter2,
    parameter3);

// 表达式换行
var isValid = condition1 &&
              condition2 &&
              condition3;
```

## 6. 文件头部

每个文件应包含文件头注释：

```csharp
/// <summary>
/// 文件说明
/// </summary>
/// <created>2026-04-30</created>
/// <author>作者</author>
```
