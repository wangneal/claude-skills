# 异常处理规范 (Exception Handling Standards)

本文档定义金蝶苍穹开发的异常处理标准。

## 1. 异常捕获

### 基本原则

- 只捕获你能处理的异常
- 记录异常信息
- 不要吞掉异常

### ✅ 正确示例

```csharp
try
{
    DynamicObject bill = LoadBill(billNo);
    ProcessBill(bill);
}
catch (KDException ex)
{
    // 记录异常详细信息
    Logger.Error($"处理单据失败: {billNo}", ex);

    // 向上抛出或处理
    throw new BusinessException("单据处理失败，请联系管理员", ex);
}
```

### ❌ 错误示例

```csharp
try
{
    ProcessBill(billNo);
}
catch (Exception ex)
{
    // 吞掉异常，不记录
}
```

## 2. 异常类型

### 使用具体的异常类型

```csharp
// ✅ 正确 - 使用具体异常
catch (ArgumentNullException ex)
{
    // 处理参数为空
}
catch (InvalidOperationException ex)
{
    // 处理操作无效
}

// ❌ 错误 - 捕获所有异常
catch (Exception ex)
{
    // 无法区分异常类型
}
```

### 自定义异常

```csharp
/// <summary>
/// 业务异常
/// </summary>
public class BusinessException : Exception
{
    public string ErrorCode { get; set; }

    public BusinessException(string message)
        : base(message)
    {
    }

    public BusinessException(string message, Exception innerException)
        : base(message, innerException)
    {
    }
}
```

## 3. 异常日志

### 日志内容

- 异常类型
- 异常消息
- 堆栈跟踪
- 关键参数值
- 时间戳

### 示例

```csharp
catch (Exception ex)
{
    Logger.Error($@"
处理单据异常:
- 单据编号: {billNo}
- 单据ID: {billId}
- 操作人: {currentUser}
- 异常类型: {ex.GetType().Name}
- 异常消息: {ex.Message}
- 堆栈: {ex.StackTrace}",
    ex);
}
```

## 4. 异常处理模式

### 模式 1: 记录并抛出

```csharp
try
{
    SaveData(data);
}
catch (Exception ex)
{
    Logger.Error("保存数据失败", ex);
    throw; // 保留原始堆栈
}
```

### 模式 2: 包装异常

```csharp
try
{
    CallExternalService();
}
catch (TimeoutException ex)
{
    throw new ServiceException("调用外部服务超时", ex);
}
```

### 模式 3: 恢复处理

```csharp
try
{
    SendEmail(message);
}
catch (SmtpException ex)
{
    Logger.Warn($"邮件发送失败: {ex.Message}");
    // 保存到重试队列
    SaveToRetryQueue(message);
}
```

## 5. Finally 块

### 何时使用

- 释放资源
- 清理临时状态
- 确保执行清理代码

### 示例

```csharp
Stream stream = null;
try
{
    stream = File.OpenRead(filePath);
    ProcessFile(stream);
}
catch (IOException ex)
{
    Logger.Error($"读取文件失败: {filePath}", ex);
    throw;
}
finally
{
    if (stream != null)
    {
        stream.Dispose();
    }
}
```

### 使用 using 语句（推荐）

```csharp
using (var stream = File.OpenRead(filePath))
{
    ProcessFile(stream);
}
```

## 6. 常见场景

### 数据库操作

```csharp
try
{
    using (var conn = new SqlConnection(connectionString))
    {
        conn.Open();
        ExecuteNonQuery(conn, sql);
    }
}
catch (SqlException ex)
{
    Logger.Error($"数据库操作失败: {ex.Message}", ex);
    throw new DataException("数据操作失败", ex);
}
```

### 工作流操作

```csharp
try
{
    WorkflowServiceHelper.SubmitToWorkflow(billNo, billId);
}
catch (KDWorkflowException ex)
{
    Logger.Error($"工作流提交失败: {billNo}", ex);
    throw new BusinessException("流程提交失败，请检查流程配置", ex);
}
```
