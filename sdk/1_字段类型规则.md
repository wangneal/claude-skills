# 一、字段类型推断规则

> GetValue 返回什么类型？一看字段名就知道。


## 1.1 普通字段（返回基本类型）


| 字段特征 | 示例 | 返回类型 | 取值方式 |

|----------|------|----------|----------|

| FAmount/Qty/Price/Discount | FAmount | decimal | Convert.ToDecimal(obj["FAmount"]) |

| FDate/BizDate | FBizDate | DateTime | Convert.ToDateTime(obj["FBizDate"]) |

| FNote/Text/Description | FNote | string | obj["FNote"].ToString() |

| FDocumentStatus | FDocumentStatus | string | obj["FDocumentStatus"].ToString() |

| FIs开头 | FIsvoid | bool | Convert.ToBoolean(obj["FIsvoid"]) |


## 1.2 基础资料字段（返回 DynamicObject）


| 字段特征 | 示例 | 取值方式 |

|----------|------|----------|

| F{Entity}Id (Id结尾) | FMaterialId | as DynamicObject 后取属性 |

| F{Entity}Number | FMaterialNumber | (obj["FMaterialId"] as DynamicObject)?["FNUMBER"].ToString() |

| F{Entity}Name | FMaterialName | (obj["FMaterialId"] as DynamicObject)?["FNAME"].ToString() |


```csharp
DynamicObject material = this.EntityObject["FMaterialId"] as DynamicObject;
string materialNumber = material["FNUMBER"].ToString();
string materialName = material["FNAME"].ToString();
```


## 1.3 单据体字段（返回 DynamicObjectCollection）


| 字段特征 | 示例 | 取值方式 |

|----------|------|----------|

| F{Entity}Entry | FSaleOrderEntry | as DynamicObjectCollection 后遍历 |


```csharp
DynamicObjectCollection entries = this.EntityObject["FSaleOrderEntry"] as DynamicObjectCollection;
foreach (DynamicObject row in entries)
{
    decimal qty = Convert.ToDecimal(row["FQTY"]);
}
```
