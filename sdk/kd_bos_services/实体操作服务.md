# 实体操作服务

> 来源: https://dev.kingdee.com/open/detail/sdk/2377816510149179392

1. 实体操作服务介绍

实体操作服务类（kd.bos.servicehelper.operation.OperationServiceHelper），主要提供单据操作服务相关接口。
SDK：https://dev.kingdee.com/sdk/Cosmic%20V5.0.002/index.html?nav=class

1.1 方法介绍
序号	方法名	方法说明	参数说明
1	executeOperate(String operationKey, String entityNumber, DynamicObject[] dataEntities, OperateOption option)	调用操作服务	operationKey 操作代码
entityNumber 实体标识
dataEntities 需要操作的数据
option 自定义参数
2	executeOperate(String operationKey, String entityNumber, Object[] ids, OperateOption option)	调用操作服务	operationKey 操作代码
entityNumber 实体标识
ids 数据实体的id（主键）
option 自定义参数
1.2 方法用例

简单介绍相关方法调用。

1.2.1 1）、executeOperate(String operationKey, String entityNumber, DynamicObject[] dataEntities, OperateOption option)

基于传入的数据包执行操作。

调用示例：

DynamicObject dataEntity = this.getModel().getDataEntity();
OperationServiceHelper.executeOperate("save", "test_entity", new DynamicObject[]{dataEntity}, OperateOption.create());
1.2.2 2）、executeOperate(String operationKey, String entityNumber, Object[] ids, OperateOption option)

基于单据内码执行操作，操作执行时，会自行到数据库读取单据数据，然后再基于数据包执行操作。

调用示例：

BillList grid = this.getView().getControl("billlistap");
//获取选择行数据
ListSelectedRowCollection col = grid.getSelectedRows();
Object[] ids = new Object[col.size()];
int i = 0;
for (ListSelectedRow obj : col) {
    ids[i++] = obj.getPrimaryKeyValue();
}
DynamicObject dataEntity = this.getModel().getDataEntity();
OperationServiceHelper.executeOperate("save", "test_entity", ids, OperateOption.create());
